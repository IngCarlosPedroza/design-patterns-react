<h1>Layout Components</h1>

<p>Layout components are components that deal primarily with arranging other components on the page.</p>

<p>Layout components should't know where they are being displayed.</p>

<h2>Examples</h2>

<ul>
  <li>Split screen</li>
  <li>Item</li>
  <li>List</li>
  <li>Modal</li>
</ul>
